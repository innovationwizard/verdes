name: Quality grades
description: Cardamom quality grades in three tiers - all can have zero or positive kilos per shipment
type: project
---

Three distinct tiers:
- **Top (SGQ):** P1, P2, P3
- **Medium (PGQ):** S1 +7mm, S2 5-6mm
- **Low (MYQ/other):** GOP, YELLOW (LG), MYQ, TRIP, ORO

All grades can have 0 or positive kilos in any given shipment. None should be hidden — all must always be available for allocation.
